# Animal Charity Website 
This is my Animal charity web application.

## About Project
I made a website for a  animal charity organization called "Paws and Hearts".
The website shows their mission, programs and donation page.

## Technologies Used
- HTML
- CSS
- JavaScript
- Font Awesome (for icons)

## Pages / Sections
- Home
- About Us
- Programs
- Animals (adoption)
- Donate
- Contact

## Features
- Navbar with hamburger menu for mobile
- Animated counter on homepage
- Donation form with validation
- Contact form
- Scroll to top button
- Responsive design (works on mobile too)
- 

## What I Learned
- How to make responsive navbar
- JavaScript form validation
- CSS animations and hover effects
- How to use Font Awesome icons
- Flexbox and Grid layout
  
## Purpose of Project
The main purpose of this project is to spread awareness about animal care and charity.
